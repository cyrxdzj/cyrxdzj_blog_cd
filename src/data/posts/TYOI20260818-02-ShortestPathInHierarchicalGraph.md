### 什么是分层图最短路

分层图最短路用于解决**允许进行有限次特殊操作**的最短路问题。例如，可以免费乘坐 $k$ 次航线、可以将 $k$ 条边的权值减半、可以跳过 $k$ 条边等。

核心思想是将原图复制成 $k+1$ 层（第 $0$ 层到第 $k$ 层），其中第 $i$ 层表示**已经使用了 $i$ 次特殊操作**时的状态。在层内连原图的边（表示不使用特殊操作），在层间连代表使用一次特殊操作的边（权值取决于具体操作，如免费则为 $0$，减半则为 $w/2$ 等）。这样跑最短路时，从第 $0$ 层到第 $k$ 层的过程就对应了使用操作次数的变化。最终答案取所有层中终点的最短距离，即 $\min\limits_{0 \le i \le k} dist[t + i \times n]$（或 $\min\limits_{0 \le i \le k} dist[i][t]$）。

### 建图方式

分层图最短路有两种常见的实现方式：

**方式一：显式建分层图（扩容法）**

将第 $i$ 层的节点 $v$ 映射为全局编号 $v + i \times n$。层间边为有向边，方向从第 $i$ 层指向第 $i+1$ 层。节点总数为 $(k+1) \times n$，每条原图的边在每层都有一条副本，层间还需建对应的跨层边，总边数约 $2km$。注意 $k$ 通常较小（如 $\le 100$），否则显式建图可能超内存。建好大图后直接跑最短路。

**方式二：隐式建图（DP法）**

不实际新建节点，而是用二维数组 $dist[i][v]$ 表示已用 $i$ 次操作到达节点 $v$ 的最小代价。在 Dijkstra 或 SPFA 的松弛过程中，同时考虑同层转移和跨层转移。这种方式不需要额外建边，更节省空间。

### 例题：[Luogu P4568 【JLOI2011】 飞行路线](https://www.luogu.com.cn/problem/P4568)

#### 题目描述

给定 $n$ 个城市（编号 $0 \sim n-1$）和 $m$ 条无向航线，每条航线连接两个城市，费用为 $c$（$0 \le c \le 10^3$）。最多可以免费乘坐 $k$ 次航线。求从起点 $s$ 到终点 $t$ 的最少花费。

数据范围：$2 \le n \le 10^4$，$1 \le m \le 5 \times 10^4$，$0 \le k \le 10$。

#### 分层图建图

将图复制成 $k+1$ 层（$i = 0 \sim k$），第 $i$ 层表示已用 $i$ 次免费机会。对每条原边 $(u, v, w)$：

- **同层边**：在第 $i$ 层内，$u + i \times n$ 与 $v + i \times n$ 之间连双向边，权值为 $w$（表示不使用免费机会，正常付费）。
- **跨层边**（$i = 0 \sim k-1$）：从 $u + i \times n$ 到 $v + (i+1) \times n$ 连权值为 $0$ 的边（表示在 $u \to v$ 上使用一次免费机会），同时从 $v + i \times n$ 到 $u + (i+1) \times n$ 也连一条权值为 $0$ 的边（对应无向边的另一个方向）。

节点总数 $(k+1) \times n$，边数 $O(km)$。

#### 堆优化Dijkstra

边权均为非负，用堆优化 Dijkstra 求最短路。最终答案取 $\min\limits_{0 \le i \le k} dist[t + i \times n]$，因为不一定用满 $k$ 次机会。

#### 复杂度分析

- **时间复杂度**：$O(k(n+m) \log(kn))$。
- **空间复杂度**：$O((k+1)n + km)$。

### 例题：[Luogu P3119 【USACO15JAN】 Grass Cownoisseur G](https://www.luogu.com.cn/problem/P3119)

#### 题目描述

给定 $n$ 个草场和 $m$ 条单向牛道。Bessie 从草场 $1$ 出发，希望访问尽可能多的不同草场后回到草场 $1$。她**至多可以逆行一次**（即沿一条单向道的反方向走）。每个草场即使多次经过也只算一次。

数据范围：$1 \le n, m \le 10^5$。

#### 解题思路

本题看似可以套用分层图模型（将 DAG 复制两层，跨层连逆行边跑最长路），但这样做存在一个问题：**在分层图中，同一个 SCC 节点在两层中各自出现，最长路可能先后经过两层中的同一 SCC，导致点权被重复计入**，违背"多次经过只算一次"的题意。

正确的做法用到了分层图思想中"拆分路径阶段"的核心，但不建分层图，而是通过 DAG 上的两次 DP 来解决。

##### 1. 缩点

在同一个强连通分量（SCC）内的草场可以互相到达。Tarjan 缩点后，将 SCC 的大小作为该点的**点权**，得到一张 DAG。记 $bel[x]$ 表示原节点 $x$ 所属的 SCC 编号，$size[i]$ 表示编号为 $i$ 的 SCC 的大小。

##### 2. 两次 DP

在 DAG 上做两次拓扑排序 DP：

- **正向 DP**：设 $d1[i]$ 表示从 $bel[1]$ 出发到 $i$ 能访问到的最大点权和（含两端）。按拓扑序正向推：
  $$d1[v] = \max(d1[v],\ d1[u] + size[v])$$

- **反向 DP**：设 $d2[i]$ 表示从 $i$ 出发到 $bel[1]$ 能访问到的最大点权和（含两端）。在反图上按拓扑序推，或直接在原图上按拓扑序反向处理：
  $$d2[u] = \max(d2[u],\ d2[v] + size[u])$$
  注意 $d2[v]$ 已包含 $size[v]$，加上 $size[u]$ 后正好是从 $u$ 到 $bel[1]$ 的完整路径。

初始值：$d1[bel[1]] = d2[bel[1]] = size[bel[1]]$，其余节点初始化为 $-\infty$ 表示不可达。

##### 3. 枚举逆行边

缩点后 SCC 内部节点已通过 size 计算，无需再考虑内部边的逆行。对于原图的每条有向边 $(u, v)$，若 $bel[u] \neq bel[v]$，将其视作逆行边：

- 正向走到 $bel[v]$，逆行回到 $bel[u]$，再正向走回 $bel[1]$。
- 该方案的点权和 = $d1[bel[v]] + d2[bel[u]] - size[bel[1]]$。

$d1$ 和 $d2$ 的路径仅在 $bel[1]$ 处重叠（在 DAG 中，从 $bel[1]$ 可达的节点集与可达 $bel[1]$ 的节点集不会交叉，否则构成环），因此只需减去一次 $size[bel[1]]$。

最终答案为所有逆行边方案的最大值，并与不逆行的情况（即 $d1[bel[1]]$）取 $\max$。若全图缩点后只剩 $bel[1]$（即全图强连通），答案直接为 $size[bel[1]]$。

#### 复杂度分析

- **时间复杂度**：Tarjan 缩点 $O(n + m)$，两次拓扑序 DP $O(n + m)$，枚举边 $O(m)$。
- **空间复杂度**：$O(n + m)$。

### 例题：[Luogu P1948 【USACO08JAN】 Telephone Lines S](https://www.luogu.com.cn/problem/P1948)

#### 题目描述

给定 $n$ 个点、$p$ 条无向边，每条边有边权 $L_i$。电信公司免费提供 $k$ 条电话线，其余电话线的费用等于其中最长的电话线的长度（即求路径上第 $k+1$ 大边权的最小值）。求从 $1$ 到 $n$ 的最小花费，不连通输出 $-1$。

数据范围：$n \le 1000$，$p \le 10000$，$0 \le k < n$，$L_i \le 10^6$。

#### 解题思路

若直接使用分层图最短路，节点数 $(k+1) \times n$ 最多约 $10^6$，在可接受范围内。但题目要求"最小化路径上第 $k+1$ 大的边权"，这更适合使用**二分答案**。

二分最终花费 $X$，上界设为 $\max L_i + 1$（哨兵值），对每个 $X$ 判断是否存在 $1$ 到 $n$ 的路径，使得其中 $>X$ 的边不超过 $k$ 条：

1. **边权转换**：将原图的边分为两类——$>X$ 的边代价为 $1$，$\le X$ 的边代价为 $0$。
2. **0-1 BFS**：在新图上求 $1$ 到 $n$ 的最短路。由于边权只有 $0$ 和 $1$，用双端队列 BFS：遇到 $0$ 边将节点入队首，遇到 $1$ 边入队尾，复杂度 $O(n + p)$。
3. **判定**：若最短路 $\le k$，则 $X$ 可行，向左二分；否则向右二分。

二分完毕后若最终答案为哨兵值 $\max L_i + 1$，说明不连通，输出 $-1$。

#### 复杂度分析

- **时间复杂度**：二分 $\log(\max L_i) \approx 20$ 轮，每轮 0-1 BFS $O(n + p)$，总 $O((n + p) \log(\max L_i))$。
- **空间复杂度**：$O(n + p)$。

### 分层图最短路的一般步骤

1. **确定特殊操作**：分析题意，找出可以执行多少次什么类型的特殊操作（免费、减半、跳过等）。
2. **确定层数**：若最多执行 $k$ 次操作，则建 $k+1$ 层图。注意 $k$ 较大时（如 $\ge 10^4$ 级别）分层图节点数会爆炸，需换用二分答案等思路。
3. **确定转移方式**：
   - 同层边：维持原图的边权，表示不使用特殊操作。
   - 跨层边：通常权值为 $0$，表示使用一次特殊操作。有时也需要根据题意调整松弛条件（如取 $\max$ 而非求和）。
4. **选择最短路算法**：边权非负时优先使用堆优化 Dijkstra；有负权边或需要判负环时使用 SPFA。
5. **统计答案**：答案通常取所有层中终点的最值（$\min$ 或 $\max$），注意可能不需用满 $k$ 次操作。
