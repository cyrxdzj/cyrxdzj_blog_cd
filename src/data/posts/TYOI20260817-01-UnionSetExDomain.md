### 什么是扩展域并查集

普通并查集只能维护元素之间的等价关系——即两个元素是否属于同一集合。但在许多问题中，元素之间的关系不仅仅是“同类”或“不同类”，还可能存在捕食、敌对等更复杂的关系。

**扩展域并查集**（又称**种类并查集**）的核心思想是：将每个元素拆分为多个“域”，每个域代表该元素在不同关系下的状态，从而将复杂关系转化为并查集能够处理的等价关系。

具体来说，如果一个元素有 $k$ 种可能的状态，我们就为它创建 $k$ 个域，分别对应这 $k$ 种状态。当确定两个元素之间的某种关系时，我们便按照关系所对应的规则，合并它们相应的域。

### 域的拆分与关系映射

假设有 $n$ 个元素，每个元素有 $k$ 种状态。我们开 $k \times n$ 的数组：

- 编号 $1 \sim n$：第1种状态域
- 编号 $n+1 \sim 2n$：第2种状态域
- 编号 $2n+1 \sim 3n$：第3种状态域
- 以此类推

对于元素 $x$：
- $x$ 表示 $x$ 处于状态1
- $x + n$ 表示 $x$ 处于状态2
- $x + 2n$ 表示 $x$ 处于状态3

当我们得知“元素 $x$ 处于状态1，则元素 $y$ 必须处于状态2”时，就执行合并操作 `merge(x, y + n)`。

### 时空复杂度

设元素数量为 $n$，关系的种类数为 $k$。

空间复杂度为 $O(k \times n)$，因为需要为每个元素维护 $k$ 个域。

时间复杂度上，每次查找或合并操作的均摊复杂度为 $O(\alpha(n))$，与普通的并查集一致，其中 $\alpha(n)$ 是反阿克曼函数，可以视为常数。

### 例题

#### 1. [Luogu P2024 【NOI2001】 食物链](https://www.luogu.com.cn/problem/P2024)

**题目描述**：动物王国中有三类动物 A、B、C，这三类动物的食物链构成了有趣的环形：A 吃 B，B 吃 C，C 吃 A。现有 $N$ 个动物，以 $1 \sim N$ 编号。每个动物都是 A、B、C 中的一种，但是我们并不知道它到底是哪一种。有人用两种说法对这 $N$ 个动物所构成的食物链关系进行描述：

1. `1 X Y`：$X$ 和 $Y$ 是同类
2. `2 X Y`：$X$ 吃 $Y$

此人说 $K$ 句话，有的是真的，有的是假的。一句话满足下列三条之一时就是假话：
- $X$ 或 $Y$ 的编号大于 $N$
- $X$ 吃 $X$
- 该话与之前确定为真话的话相矛盾

要求得出假话的总数。

**思路分析**：我们将每个动物拆成三个域：

- $x$：$x$ 属于 A 类
- $x + n$：$x$ 属于 B 类
- $x + 2n$：$x$ 属于 C 类

由于 A 吃 B，B 吃 C，C 吃 A，可以推导出：

当 $X$ 和 $Y$ **是同类**时：
- 若 $X$ 是 A，则 $Y$ 也是 A，合并 `merge(x, y)`
- 若 $X$ 是 B，则 $Y$ 也是 B，合并 `merge(x + n, y + n)`
- 若 $X$ 是 C，则 $Y$ 也是 C，合并 `merge(x + 2 * n, y + 2 * n)`

当 **$X$ 吃 $Y$** 时（A 吃 B，B 吃 C，C 吃 A）：
- 若 $X$ 是 A，则 $Y$ 是 B（A 吃 B），合并 `merge(x, y + n)`
- 若 $X$ 是 B，则 $Y$ 是 C（B 吃 C），合并 `merge(x + n, y + 2 * n)`
- 若 $X$ 是 C，则 $Y$ 是 A（C 吃 A），合并 `merge(x + 2 * n, y)`

**判断假话**：

当声明“$X$ 和 $Y$ 是同类”时，若 $X$ 的任意域与 $Y$ 的不同域在同一集合中，说明 $X$ 与 $Y$ 的种类不同，该话为假。例如，若 `find(x) == find(y + n)`（$X$ 是 A 而 $Y$ 是 B），或 `find(x + n) == find(y)`（$X$ 是 B 而 $Y$ 是 A），均说明矛盾。

当声明“$X$ 吃 $Y$”时，若 $X$ 和 $Y$ 的同类域在同一集合中，或存在 $Y$ 吃 $X$ 的关系（例如 `find(x + n) == find(y)`，即 $X$ 是 B 而 $Y$ 是 A，说明 A 吃 B 即 $Y$ 吃 $X$），该话为假。
